/src
├── /assets           → Görseller, logolar, sabit dosyalar
├── /components       → Reusable UI bileşenleri (Button, Modal, Input vs.)
├── /layouts          → Sayfa şablonları (Navbar, Sidebar, AuthLayout vs.)
├── /pages            → Route edilen ana sayfalar (CustomersPage, DashboardPage)
├── /features         → Modül bazlı state & UI mantığı (customer, auth vs.)
│   ├── /customer
│   │   ├── CustomerList.jsx
│   │   ├── CustomerForm.jsx
│   │   ├── customerSlice.js (ileride Redux veya Zustand için)
├── /hooks            → Custom hook’lar (useFetch, useForm vs.)
├── /services         → API çağrıları (axios instance, api/customer.js vs.)
├── /utils            → Yardımcı fonksiyonlar (formatDate, generateId vs.)
├── /constants        → Sabit değerler (routes, regex, messages vs.)
├── /styles           → Tailwind'e özel global stiller veya varsayılanlar
├── App.jsx
├── main.jsx
